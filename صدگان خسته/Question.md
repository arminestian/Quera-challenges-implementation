# مقایسهٔ معکوس دو عدد سه‌رقمی

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

برنامه‌ای بنویسید که **دو عدد سه‌رقمی** را **به‌صورت برعکس** مقایسه کند.  
به این صورت که **ارزش یکان بیش از دهگان** و **ارزش دهگان بیش از صدگان** است.

مثال:

```

321 < 123
201 > 800

```

---

## ورودی

در دو خط:

- خط اول: عدد اول  
- خط دوم: عدد دوم  

هر دو عدد **مثبت** و **سه‌رقمی** هستند.

---

## خروجی

- عددی که **به‌صورت معکوس کوچک‌تر** است در سمت چپ چاپ شود، سپس علامت `<` و سپس عدد بزرگ‌تر.  
- اگر دو عدد در حالت معکوس **برابر** باشند، بین آن‌ها علامت `=` چاپ کنید.  
- **تمام اجزا باید با space جدا شوند.**

---

## مثال‌ها

### ورودی نمونه ۱
```

123
421

```

### خروجی نمونه ۱
```

421 < 123

```
