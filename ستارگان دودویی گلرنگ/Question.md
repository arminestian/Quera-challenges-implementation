# نمایش ستاره‌ای رشته‌های 0 و 1

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

---

ستاره‌های گلرنگ رشته‌های `0` و `1` را به روش مخصوص خود نمایش می‌دهند:

- نمایش `0` به صورت:
```

---

*.*

---

```

- نمایش `1` به صورت:
```

.*.
.*.
.*.

```

حالا یک رشته‌ی متشکل از ارقام `0` و `1` به شما داده می‌شود؛  
باید نمایش کلی آن رشته را با چسباندن نمایش هر رقم به‌صورت سِرفرصت کنار هم چاپ کنید (سه سطر خروجی).

---

## ورودی

- سطر اول: عدد صحیح مثبت `n` — طول رشته.  
```

1 ≤ n ≤ 100

```
- سطر دوم: یک رشته‌ی طول `n` شامل فقط کاراکترهای `0` و `1`.

---

## خروجی

سه سطر چاپ کنید که نمایش ستاره‌ای رشته را نشان می‌دهد (هر رقم با نمایش مربوطه جایگزین و کنار هم چسبیده شده است).

---

## مثال‌ها

### مثال ۱
**ورودی**
```

4
0101

```

**خروجی**
```

***.*.***.*.
*.*.*.*.*.*.
***.*.***.*.

```

---

### مثال ۲
**ورودی**
```

5
10011

```

**خروجی**

.*.******.*..*.
.*.*.**.*.*..*.
.*.******.*..*.
