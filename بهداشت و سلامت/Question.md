# مسئله: نمره نوروز و سفر

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

---

نوروز برای سفر برنامه‌ریزی کرده است و استادش، آقا فیروز، قوانین نمره‌دهی زیر را مشخص کرده است:

- اگر نوروز **سفر نرود** (۰ روز سفر)، نمره‌ی او **۲۰** خواهد بود.
- اگر نوروز **دقیقاً ۷ روز** سفر کند، نمره‌ی او همان نمره‌ی اولیه‌ی `X` خواهد بود.
- در هر حالت دیگر، نمره‌ی نوروز به اندازه‌ی تعداد روزهای سفر کم می‌شود (ولی حداقل نمره صفر است).

---

## ورودی

- خط اول: یک عدد صحیح `X` (نمره فعلی نوروز)  
  `0 ≤ X ≤ 20`
- خط دوم: یک عدد صحیح `N` (تعداد روزهای سفر)  
  `0 ≤ N ≤ 100`

---

## خروجی

- یک عدد صحیح — نمره نهایی نوروز در درس بهداشت و سلامت.

---

## مثال‌ها

### مثال ۱
**ورودی**
```

14
0

```
**خروجی**
```

20

```
**توضیح:** نوروز سفر نمی‌رود، بنابراین نمره‌ی او ۲۰ می‌شود.

---

### مثال ۲
**ورودی**
```

6
7

```
**خروجی**
```

6

```
**توضیح:** سفر دقیقاً ۷ روز است، بنابراین نمره همان ۶ باقی می‌ماند.

---

### مثال ۳
**ورودی**
```

13
9

```
**خروجی**
```

4

```
**توضیح:** نوروز ۹ روز سفر کرده، پس نمره‌ی او ۱۳ - ۹ = ۴ می‌شود.
