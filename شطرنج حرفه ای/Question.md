# مهره‌های ناقص پویان ♟️

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

---

پویان یک نوجوان تپل است که تصمیم گرفته با ورزش **شطرنج** وزن خود را کاهش دهد!  
او از بزرگان شطرنج یک مجموعه مهره قرض گرفته، اما وقتی مهره‌ها را شمرد، فهمید تعداد آن‌ها درست نیست —  
تعداد بعضی از مهره‌ها **بیشتر** و بعضی **کمتر** از مقدار استاندارد است.

---

## ترکیب استاندارد مهره‌های شطرنج
یک مجموعه کامل باید شامل موارد زیر باشد:

| مهره | تعداد استاندارد |
|------|------------------|
| شاه | ۱ |
| وزیر | ۱ |
| رخ | ۲ |
| فیل | ۲ |
| اسب | ۲ |
| سرباز | ۸ |

---

## وظیفه شما
با گرفتن تعداد فعلی هر نوع مهره که پویان دارد، مشخص کنید او باید برای هر نوع مهره:
- **چند عدد تهیه کند** (اگر کم دارد)، یا  
- **چند عدد حذف کند** (اگر زیاد دارد).

---

## ورودی
در تنها خط ورودی، **۶ عدد صحیح** آمده که به ترتیب نشان‌دهنده‌ی تعداد فعلی مهره‌های پویان هستند:

```

king queen rook bishop knight pawn

```

---

## خروجی
در خروجی، **۶ عدد** به همان ترتیب ورودی چاپ کنید که هر عدد بیانگر این است:

- اگر مقدار مثبت باشد → پویان باید آن‌قدر **مهره اضافه کند**.  
- اگر مقدار منفی باشد → پویان باید آن‌قدر **مهره حذف کند**.  
- اگر مقدار صفر باشد → نیازی به تغییر ندارد.

---

## مثال

### ورودی نمونه
```

0 2 2 2 3 7

```

### خروجی نمونه
```

1 -1 0 0 -1 1

```

**توضیح:**  
- شاه: باید ۱ عدد اضافه شود.  
- وزیر: باید ۱ عدد حذف شود.  
- رخ، فیل: کامل است.  
- اسب: باید ۱ عدد حذف شود.  
- سرباز: باید ۱ عدد اضافه شود.
