# تقسیم کیک‌های کوئرا

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۶۴ مگابایت  

در شرکت کوئرا، ۳ کیک هم‌اندازه‌ی دایره‌ای برای تولد باقر خریداری شده است.

خانم عبادی می‌تواند هر کیک را به یکی از ۳ روش زیر تقسیم کند:

1. کیک را **کامل** به ۱ نفر بدهد.  
2. کیک را **دو قسمت** کند و به ۲ نفر بدهد.  
3. کیک را **چهار قسمت** کند و به ۴ نفر بدهد.

**توجه:**  
برای *هر کیک* دقیقاً **یکی** از این سه عملیات انجام می‌شود.  
برای مثال نمی‌توان یک نصفه کیک را دوباره نصف کرد.

در روز تولد باقر، `n` نفر در شرکت حضور دارند.  
به خانم عبادی کمک کنید تعیین کند آیا می‌توان با انتخاب یکی از سه روش بالا برای هر کیک، کاری کرد که در نهایت **به همه‌ی n نفر مقدار برابری کیک برسد** یا خیر.

---

## ورودی

در تنها خط ورودی:

```

n

```

عدد صحیح مثبت:

```

1 ≤ n ≤ 12

```

---

## خروجی

در تنها خط خروجی:

- اگر چنین تقسیم‌بندی‌ای **ممکن است**، چاپ کنید: `YES`  
- در غیر این صورت چاپ کنید: `NO`

> **سیستم داوری نسبت به حروف کوچک و بزرگ حساس است.**

---

## مثال‌ها

### مثال ۱

**ورودی**
```

1

```

**خروجی**
```

YES

```

**توضیح:**  
هر سه کیک را به همان یک نفر می‌دهیم.

---

### مثال ۲

**ورودی**
```

2

```

**خروجی**
```

YES

```

**توضیح:**  
- کیک اول → نفر اول  
- کیک دوم → نفر دوم  
- کیک سوم → نصف شده → نیم‌کی به نفر اول و نیم‌کی به نفر دوم

---

### مثال ۳

**ورودی**
```

3

```

**خروجی**
```

YES

```

**توضیح:**  
هر نفر یک کیک می‌گیرد.

---

### مثال ۴

**ورودی**
```

11

```

**خروجی**
```

NO`
