# دنبالهٔ فیبوناچی و علامت‌گذاری

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

---

دنبالهٔ فیبوناچی به‌صورت زیر تعریف می‌شود:

```

fib(1) = 1
fib(2) = 2
fib(n) = fib(n - 1) + fib(n - 2)

```

اکنون برنامه‌ای بنویسید که با دریافت عدد طبیعی `n`،  
رشته‌ای شامل کاراکترهای `'+'` و `'-'` چاپ کند به طوری که:

- اگر عدد `i` جزو دنبالهٔ فیبوناچی باشد، `sᵢ = '+'`
- در غیر این صورت، `sᵢ = '-'`

---

## ورودی
در تنها خط ورودی عدد طبیعی `n` داده می‌شود:

```

1 ≤ n ≤ 100

```

---

## خروجی
در تنها خط خروجی رشته‌ای به طول `n` چاپ کنید  
که مطابق توضیحات بالا ساخته شده باشد.

---

## مثال‌ها

### مثال ۱
**ورودی**
```

15

```
**خروجی**
```

+++-+--+----+--

```
