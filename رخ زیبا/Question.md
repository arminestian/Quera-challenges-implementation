## مسئله: شطرنج و خوش‌حالی عرفان

عرفان و متین شطرنج بازی می‌کنند. شرایط مسئله:

- مهره‌های **رخ عرفان** در خانه‌های `(x1, y1)` و `(x2, y2)` هستند.
- مهره‌های **رخ متین** در خانه‌های `(x3, y3)` و `(x4, y4)` هستند.
- هیچ دو مهره در یک خانه قرار ندارند.

**قواعد تهدید رخ:**

- رخ‌ها می‌توانند تمام خانه‌های هم‌سطر و هم‌ستون خود را تهدید کنند.
- رخ عرفان "زیبا" است اگر حداقل یک رخ متین را تهدید کند.
- عرفان خوش‌حال است اگر **دقیقا یک** رخش زیبا باشد.

---

### ورودی

چهار خط، هر خط شامل مختصات یک مهره `(x, y)`:

```

1 ≤ xi, yi ≤ 8

```

ترتیب خطوط:

1. رخ اول عرفان `(x1, y1)`
2. رخ دوم عرفان `(x2, y2)`
3. رخ اول متین `(x3, y3)`
4. رخ دوم متین `(x4, y4)`

---

### خروجی

- `happy` اگر دقیقا یک رخ عرفان زیبا باشد
- `unhappy` در غیر این صورت

---

### نمونه‌ها

**ورودی نمونه ۱**
```

1 1
2 2
3 3
4 4

```

**خروجی نمونه ۱**
```

unhappy

```

> توضیح: هیچ کدام از رخ‌های عرفان، رخ‌های متین را تهدید نمی‌کنند.

---

**ورودی نمونه ۲**
```

1 1
1 2
1 3
8 2

```

**خروجی نمونه ۲**
```

unhappy

```

> توضیح: رخ اول عرفان رخ اول متین را تهدید می‌کند، رخ دوم هر دو رخ متین را تهدید می‌کند. هر دو رخ عرفان زیبا هستند → عرفان خوش‌حال نیست.

---

**ورودی نمونه ۳**
```

2 1
4 3
2 7
8 6

```

**خروجی نمونه ۳**
```

happy

```

> توضیح: رخ اول عرفان رخ اول متین را تهدید می‌کند → زیبا، رخ دوم هیچ رخ متین را تهدید نمی‌کند. دقیقا یک رخ زیبا → عرفان خوش‌حال است.
