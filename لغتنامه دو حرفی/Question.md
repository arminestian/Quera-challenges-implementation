# آخرین کاراکتر رشتهٔ n‌ام

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

تمام رشته‌های ساخته‌شده با حروف **a** و **b** را ابتدا بر اساس **تعداد کاراکترها** و سپس **به‌ترتیب لغت‌نامه‌ای** مرتب کرده‌ایم.  
ابتدای این فهرست به‌صورت زیر است:

| شماره | رشته |
|------|-------|
| 1    | a     |
| 2    | b     |
| 3    | aa    |
| 4    | ab    |
| 5    | ba    |
| 6    | bb    |
| 7    | aaa   |
| 8    | aab   |
| 9    | aba   |
| 10   | abb   |
| 11   | baa   |
| 12   | bab   |
| 13   | bba   |
| 14   | bbb   |
| 15   | aaaa  |
| 16   | aaab  |
| …    | …     |

حال، عدد **n** به شما داده می‌شود و باید **آخرین کاراکتر رشتهٔ n‌ام** را خروجی دهید.

---

## ورودی

در یک سطر: عدد صحیح و مثبت **n**

```

1 ≤ n ≤ 100

```

---

## خروجی

در یک سطر، **آخرین کاراکتر رشتهٔ n‌ام** را چاپ کنید.

---

## مثال‌ها

### ورودی نمونه ۱
```

3

```

### خروجی نمونه ۱
```

a

```
رشتهٔ سوم **aa** است و حرف آخر آن **a** است.

---

### ورودی نمونه ۲
```

16

```

### خروجی نمونه ۲
```

b

```
رشتهٔ شانزدهم **aaab** است و حرف آخر آن **b** است.
