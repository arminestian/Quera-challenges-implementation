## مسئله: کمینه کردن جیغ دارا و سارا

دارا و سارا خواهر و برادر هستند و در مراسم سال نو مادربزرگ برایشان `n` عروسک خریده است.  
اندازه‌ی این عروسک‌ها از 1 تا n است، که 1 کوچک‌ترین و n بزرگ‌ترین عروسک است.

### قواعد جیغ زدن:

- دارا عاشق عروسک‌های بزرگ است. اگر عروسکی **بزرگ‌تر از همه عروسک‌های قبلی** را ببیند، جیغ می‌زند.
- سارا عاشق عروسک‌های کوچک است. اگر عروسکی **کوچک‌تر از همه عروسک‌های قبلی** را ببیند، جیغ می‌زند.
- پس اولین عروسک همیشه باعث دو جیغ می‌شود.

مادربزرگ می‌خواهد ترتیب نمایش عروسک‌ها را طوری انتخاب کند که **مجموع تعداد جیغ‌ها کمینه شود**.

---

### ورودی

یک خط شامل عدد صحیح و مثبت `n` که تعداد عروسک‌ها را نشان می‌دهد.

```

1 ≤ n ≤ 100

```

---

### خروجی

یک خط شامل **کمینه مجموع تعداد جیغ‌های دارا و سارا**.

---

### نمونه ورودی و خروجی

**ورودی نمونه ۱**
```

1

```

**خروجی نمونه ۱**
```

2

```
> تنها یک عروسک وجود دارد، بنابراین دارا و سارا هر دو جیغ می‌زنند.

---

**ورودی نمونه ۲**
```

2

```

**خروجی نمونه ۲**
```

3

```
> بهترین ترتیب:
> 1. عروسک 1 → هر دو جیغ می‌زنند (2 جیغ)
> 2. عروسک 2 → دارا جیغ می‌زند (1 جیغ)
> مجموع جیغ‌ها = 3

---

**ورودی نمونه ۳**
```

3

```

**خروجی نمونه ۳**
```

3

```
> بهترین ترتیب:
> 1. عروسک 1 → هر دو جیغ می‌زنند (2 جیغ)
> 2. عروسک 3 → دارا جیغ می‌زند (1 جیغ)
> 3. عروسک 2 → هیچ جیغی نیست (0 جیغ)
> مجموع جیغ‌ها = 3
```
