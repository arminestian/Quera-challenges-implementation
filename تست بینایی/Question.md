# تست بینایی هنرجویان

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

محمدرضاص که کنکورش را داده، می‌خواهد در همه‌ی مسابقات برنامه‌نویسی کوئرا شرکت کند؛ اما اکنون درگیر گرفتن تست بینایی از هنرجویان متقاضی گواهینامه‌ی رانندگی است.

محمدرضاص در فاصله‌ی دو متری هنرجو یک کلمه از حروف انگلیسی قرار می‌دهد و هنرجو باید از روی آن *عیناً* بنویسد. سپس تعداد حروف اشتباه نوشته‌شده را به‌عنوان مقدار کوری این فرد به او اعلام می‌کند.

او برای دادن مسابقه عجله دارد و می‌خواهد این کار را کامپیوتری انجام دهد؛ بنابراین از شما می‌خواهد برنامه‌ای بنویسید که با دریافت کلمه‌ی صحیح و کلمه‌ی نوشته‌شده توسط هنرجو، تعداد اشتباه‌ها را محاسبه کند.

---

## ورودی

- در سطر اول: عدد صحیح  
  `n` — طول کلمات.

- در سطر دوم: رشته‌ای از حروف کوچک و بزرگ انگلیسی — کلمه‌ی جلوی هنرجو.

- در سطر سوم: رشته‌ای از حروف کوچک و بزرگ انگلیسی — کلمه‌ی نوشته شده توسط هنرجو.

```

1 ≤ n ≤ 100000

```

---

## خروجی

تنها سطر خروجی باید شامل یک عدد صحیح نامنفی باشد:  
**تعداد حروفی که هنرجو اشتباه نوشته است.**

---

## مثال

**ورودی نمونه ۱**
```

3
ABC
aBD

```

**خروجی نمونه ۱**
```

2

```

**توضیح:** هنرجو حرف اول و سوم را اشتباه نوشته است.
