# اکوی سخنرانی 🎤

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

---

محمدجواد که پشت‌کار زیادی دارد، تصمیم گرفته در سخنرانی‌ای درباره‌ی پشت‌کار شرکت کند و آن را یادداشت نماید.  
اما متأسفانه مسئولین صدا، **اکو** را در بالاترین حد ممکن قرار داده‌اند! 😵‍💫

---

## توضیح عملکرد اکو
به ازای هر **کلمه‌ی n حرفی** که سخنران می‌گوید،  
بلندگو **n بار** آن را پخش می‌کند، به طوری که:

- در هر بار پخش، یک حرف از ابتدای کلمه حذف می‌شود،
- سپس به جای آن، **حرف بعدی آن** جایگزین می‌شود.

---

### مثال
اگر سخنران بگوید:

```

golabi

```

بلندگو صدا را این‌طور پخش می‌کند:

```

golabi
oolabi
lllabi
aaaabi
bbbbbi
iiiiii

```

---

## ورودی
در تنها خط ورودی، یک رشته (کلمه) می‌آید که سخنران گفته است.  
طول رشته را با `n` نشان می‌دهیم.

```

1 ≤ n ≤ 20

```

---

## خروجی
خروجی شامل `n` خط است که نشان‌دهنده‌ی صداهایی است که از بلندگو پخش می‌شود.

---

## مثال‌ها

### ورودی نمونه ۱
```

golabi

```

### خروجی نمونه ۱
```

golabi
oolabi
lllabi
aaaabi
bbbbbi
iiiiii

```

---

codecup
