محدودیت زمان: ۱ ثانیه  
محدودیت حافظه: ۲۵۶ مگابایت

برنامه‌ای بنویسید که **۴ عدد صحیح** از ورودی بگیرد و خروجی شامل **جمع، میانگین، حاصل‌ضرب، بیشینه و کمینه** این اعداد باشد.

---

## ورودی
هر یک از چهار خط شامل یک عدد صحیح است.  
محدوده:  
```

|عدد| ≤ 1000

```

---

## خروجی
پنج خط، هر خط شامل:  
```

نام خروجی : عدد با ۶ رقم اعشار

```

ترتیب خروجی‌ها:  
1. جمع اعداد (`Sum`)  
2. میانگین اعداد (`Average`)  
3. حاصل‌ضرب اعداد (`Product`)  
4. بیشینه عددها (`MAX`)  
5. کمینه عددها (`MIN`)  

---

## مثال‌ها

### ورودی نمونه
```

1
2
3
4

```

### خروجی نمونه
```

Sum : 10.000000
Average : 2.500000
Product : 24.000000
MAX : 4.000000
MIN : 1.000000
