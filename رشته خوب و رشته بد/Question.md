# ساختن رشته با زیررشته‌ها

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

---

حسینقلی دو رشتهٔ `s` و `t` دارد که هر دوی آن‌ها از حروف کوچک انگلیسی تشکیل شده‌اند.  
او رشتهٔ `s` را بسیار دوست دارد و از رشتهٔ `t` متنفر است.  
حسینقلی به دنبال یک رشتهٔ طول `n` است طوری که:

- این رشته فقط از حروف کوچک انگلیسی تشکیل شده باشد.
- `s` **زیررشته** (substring) آن باشد.
- `t` **زیررشته** آن **نباشد**.

رشتهٔ `a` زیررشتهٔ رشتهٔ `b` است اگر و تنها اگر یک بازهٔ متوالی از `b` وجود داشته باشد که برابر `a` باشد.

اگر چنین رشته‌ای وجود دارد، یکی از آن‌ها را چاپ کنید؛ در غیر این صورت چاپ کنید `-1`.

---

## ورودی

- خط اول: رشتهٔ `s`.  
- خط دوم: رشتهٔ `t`.  
- خط سوم: عدد صحیح `n` — طول رشتهٔ مورد نیاز.

قیدها:
```

1 ≤ |s|, |t| ≤ 1000
1 ≤ n ≤ 100000

```

تمام رشته‌ها از حروف کوچک انگلیسی تشکیل شده‌اند.

---

## خروجی

در تنها خط خروجی:

- اگر وجود دارد، یک رشتهٔ طول `n` شامل حروف کوچک انگلیسی چاپ کنید که `s` زیررشتهٔ آن باشد و `t` زیررشتهٔ آن نباشد.  
- در غیر این صورت یک خط حاوی `-1` چاپ کنید.

اگر چند پاسخ معتبر وجود دارد، چاپ هر یک از آن‌ها صحیح است.

---

## مثال‌ها

### نمونهٔ ورودی ۱
```

code
hamkaran
7

```

### نمونهٔ خروجی ۱
```

hamcode

```

### نمونهٔ ورودی ۲
```

hamcode
code
10

```

### نمونهٔ خروجی ۲
```

-1

```

**توضیح:**  
در نمونهٔ اول رشتهٔ `hamcode` طول 7 دارد، شامل `s = "code"` هست و چون `t = "hamkaran"` زیررشتهٔ آن نیست، جواب معتبر است.  
در نمونهٔ دوم هیچ رشتهٔ طول 10 وجود ندارد که شامل `s = "hamcode"` باشد و در عین حال `t = "code"` را نداشته باشد — بنابراین خروجی `-1` است.
