# رنگ‌های مکمل RGB

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت  

---

مهیار به ست‌کردن رنگ لباس‌ها و وسایلش اهمیت زیادی می‌دهد.  
او می‌داند رنگ‌های مکمل ست‌های قشنگی می‌سازند، اما نمی‌تواند با داشتن یک رنگ، رنگ مکمل آن را تشخیص دهد.

هر رنگ با ترکیب سه رنگ اصلی **قرمز (Red)**، **سبز (Green)** و **آبی (Blue)** ساخته می‌شود.  
میزان هرکدام از این رنگ‌ها عددی بین `0` تا `255` است.  
پس هر رنگ را می‌توان با سه‌تایی مرتب زیر نشان داد:

```

(r, g, b)

```

که:
```

0 ≤ r, g, b ≤ 255

```

---

## تعریف رنگ مکمل
دو رنگ  
```

(r₁, g₁, b₁)
(r₂, g₂, b₂)

```
را مکمل می‌نامیم اگر و تنها اگر:

```

r₁ + r₂ = 255
g₁ + g₂ = 255
b₁ + b₂ = 255

```

---

## نحوه نمایش رنگ‌ها
برای نمایش هر رنگ در قالب **کد RGB**،  
مقادیر `r` و `g` و `b` را به **مبنای ۱۶ (هگزادسیمال)** تبدیل می‌کنیم،  
به‌طوری‌که هر مقدار دقیقاً با دو رقم نوشته شود (در صورت نیاز با صفر در ابتدا).  
سپس همه را به‌صورت زیر در کنار هم می‌نویسیم:

```

#RRGGBB

```

---

## ورودی
در سطر اول عدد صحیح مثبت `t` داده می‌شود، که تعداد رنگ‌ها را نشان می‌دهد.

```

1 ≤ t ≤ 10,000

```

در هر یک از `t` سطر بعدی، یک کد رنگ به فرم `#RRGGBB` داده می‌شود  
که هر رقم آن می‌تواند بین `0-9` یا `A-F` باشد.

---

## خروجی
برای هر رنگ ورودی، کد رنگ مکمل آن را در یک خط چاپ کنید.

---
