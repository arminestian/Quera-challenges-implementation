# تعداد کل نفس‌ها در دورهٔ ۱۰۲۸

**محدودیت زمان:** ۲ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

---

در دورهٔ ۱۰۲۸ ایا، هر سوال دارای یک عدد **نَفَس‌گیری** است. اگر سؤالِ شمارهٔ `i` مقدار نفَس‌گیری‌اش برابر `a_i` باشد، هر کسی که آن سؤال را حل کند در طی فرایند حل، `a_i` بار نفس می‌گیرد. همچنین آمار نشان می‌دهد که سؤالِ `i` را `b_i` نفر حل کرده‌اند.

وظیفهٔ شما این است که مجموع تعداد دفعاتی را که در کل برای همهٔ سوال‌ها نفس کشیده شده است محاسبه کنید.

---

## ورودی

- خط اول: عدد صحیح `n` — تعداد سوال‌ها.  
- خط دوم: `n` عدد صحیح `a_1 a_2 ... a_n` که `a_i` مقدار نفَس‌گیری سؤال `i` است.  
- خط سوم: `n` عدد صحیح `b_1 b_2 ... b_n` که `b_i` تعداد افرادی است که سؤال `i` را حل کرده‌اند.

محدودیت‌ها:
```

1 ≤ n, a_i, b_i ≤ 100

```

---

## خروجی

- یک عدد صحیح — مجموع تعداد دفعاتی که در کل نفس‌ها گرفته شده‌اند.  
  (یعنی جمع روی همهٔ سوال‌ها از `a_i * b_i`)

---

## مثال‌ها

### مثال ۱
**ورودی**
```

2
1 2
3 4

```
**خروجی**
```

11

```
**توضیح:** مجموع = `1*3 + 2*4 = 3 + 8 = 11`.

---

### مثال ۲
**ورودی**
```

3
3 1 4
8 3 3

```
**خروجی**
```

39

```
**توضیح:** مجموع = `3*8 + 1*3 + 4*3 = 24 + 3 + 12 = 39`.
