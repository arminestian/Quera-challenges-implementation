## مسئله: لگاریتم پایه ۲

مش‌رجب که ۶ ساله است می‌خواهد لگاریتم پایه ۲ یک عدد طبیعی را محاسبه کند، اما جواب باید **به پایین رُند شود**.

### توضیح

لگاریتم پایه ۲ عدد `x`، عدد `y` است که:

```

2^y = x

```

ورودی عدد صحیح `n` داده می‌شود و خروجی **بخش صحیح لگاریتم پایه ۲ آن** است.

---

### ورودی

یک خط شامل عدد صحیح `n`:

```

1 ≤ n < 2^30

```

---

### خروجی

یک عدد صحیح: حاصل لگاریتم پایه ۲ به پایین رُند شده.

---
