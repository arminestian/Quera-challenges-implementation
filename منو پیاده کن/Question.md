# محاسبه مقدار result

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۶۴ مگابایت  

---

با دریافت مقدار `n` و اجرای برنامه‌ی زیر، مقدار `result` را به‌دست آورید:

```plaintext
result = 0
for i from 1 to n:
    for j from 1 to i:
        result = result + 1
````

---

## ورودی

در تنها سطر ورودی، عدد صحیح و مثبت `n` داده می‌شود.

```
1 ≤ n ≤ 10^9
```

---

## خروجی

در تنها سطر خروجی، عدد صحیح و مثبت `result` را چاپ کنید.

---

## مثال‌ها

### ورودی نمونه ۱

```
1
```

### خروجی نمونه ۱

```
1
```
