محدودیت زمان: ۱ ثانیه  
محدودیت حافظه: ۲۵۶ مگابایت

می‌دانیم **سیکل گرفتن** جایگاه ویژه‌ای در میان اهالی برره دارد.

نظام، شیرفرهاد و کیوون می‌خواهند مدرک سیکل خود را بگیرند؛ برای این کار باید در امتحان تستی‌ای که کَیانوش برای آن‌ها طراحی کرده شرکت کنند.

از آن‌جایی که این سه نفر خواندن و نوشتن بلد نیستند (!) تصمیم می‌گیرند بدون خواندن سؤالات با الگوی تکرارشوندهٔ خاصی به تست‌ها پاسخ دهند. امتحانی که کَیانوش طراحی کرده دارای `N` سؤال ۳ گزینه‌ای است. در جدول زیر الگوی پاسخ‌دهی هر فرد (دورهٔ تناوب و ادامهٔ الگو) آمده است:

- **keyvoon (کیوون)** — دورهٔ تناوب ۶:  
  `..., 2, 2, 1, 1, 3, 3, 2, 2, 1, 1, 3, 3, ...`

- **nezam (نظام)** — دورهٔ تناوب ۳:  
  `..., 3, 2, 1, 3, 2, 1, 3, 2, 1, ...`

- **shir farhad (شیرفرهاد)** — دورهٔ تناوب ۴:  
  `..., 3, 2, 1, 2, 3, 2, 1, 2, ...`

با داشتن تعداد سؤال‌ها و کلید آزمون، شما باید بیشترین نمره‌ای که هر یک از این سه نفر می‌تواند در آزمون کسب کند را محاسبه کنید و اسامی افرادی که به آن بیشترین نمره رسیده‌اند را (به ترتیب حروف الفبا) چاپ کنید.

---

## ورودی
- سطر اول: عدد صحیح `N` — تعداد سؤالات.  
- سطر دوم: یک رشته به طول `N` شامل اعداد `1` تا `3` که کلید سؤالات را مشخص می‌کند (عدد iام گزینهٔ درست سؤال iام است).

```

1 ≤ N ≤ 100

```

---

## خروجی
- سطر اول: بیشترین نمره‌ای که کسب شده است (تعداد پاسخ‌های درست).  
- در سطرهای بعد: اسامی افرادی که آن بیشترین نمره را کسب کرده‌اند، هر اسم در یک سطر و به **ترتیب حروف الفبا**:
  - `keyvoon`
  - `nezam`
  - `shir farhad`

---

## مثال‌ها

### نمونه ۱
**ورودی**
```

15
111323311123111

```

**خروجی**
```

7
keyvoon
nezam
shir farhad

```

### نمونه ۲
**ورودی**
```

10
1112321332

```

**خروجی**
```

3
keyvoon
nezam

```

**توضیح نمونه ۲:**  
در این مثال کیوون به سؤال‌های سوم، ششم و هشتم درست پاسخ داده؛ نظام سؤال‌های اول، هفتم و نهم؛ شیرفرهاد سؤال‌های دوم و هشتم را درست جواب داده است.
