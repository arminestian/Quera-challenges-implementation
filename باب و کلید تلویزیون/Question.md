# تغییر شبکهٔ تلویزیون باب

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

باب یک تلویزیون جدید با **n شبکه** خریده است. نام هر شبکه یک **رشته از حروف انگلیسی کوچک** است.

کنترل تلویزیون تنها **یک دکمه** دارد که شبکهٔ کنونی را به شبکهٔ بعدی تغییر می‌دهد. شماره‌گذاری شبکه‌ها از ۱ تا n است و رفتار دکمه به صورت زیر است:

- اگر تلویزیون روی شبکهٔ `i` باشد و `i < n` باشد، با فشردن دکمه به شبکهٔ `i+1` می‌رود.  
- اگر روی شبکهٔ `i = n` باشد، با فشردن دکمه به شبکهٔ **۱** بازمی‌گردد.

اگر تلویزیون در ابتدا روی شبکهٔ `x` باشد، مشخص کنید پس از **k بار** فشردن دکمه، تلویزیون نام کدام شبکه را نشان خواهد داد.

---

## ورودی
- ابتدا سه عدد `n`، `x` و `k` داده می‌شود.
- سپس در `n` خط بعد، نام شبکه‌های ۱ تا n به ترتیب آمده است.

شرایط:
```

1 ≤ n, k ≤ 100
1 ≤ x ≤ n
طول هر نام ≤ 100
نام‌ها یکتا هستند.

```

---

## خروجی
نام شبکه‌ای که تلویزیون پس از `k` بار فشردن دکمه روی آن قرار می‌گیرد را چاپ کنید.

---

## مثال

### ورودی نمونه
```

5 2 5
bob
carl
kevin
phil
tim

```

### خروجی نمونه
```

carl

```

شبکه‌ها پس از هر بار فشردن دکمه این‌گونه تغییر می‌کنند:

```

carl → kevin → phil → tim → bob → carl
