# بازی سلام دوری

**محدودیت زمان:** ۲ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

حسنی و `n−1` نفر از دوستانش دور یک دایره نشستند و شروع به انجام بازی اتل‌متل `k`توله می‌کنند. شیوه انجام بازی این جوری هست که حسنی به عنوان نفر اول می‌گوید "سلام!". بعد از آن در هر مرحله نفر `k` تا جلوتر از نفر قبلی می‌گوید "سلام!". این روال ادامه دارد تا دوباره نوبت حسنی شود و آن موقع بازی تموم می‌شود.

حالا حسنی می‌خواهد بداند که این بازی چند مرحله طول می‌کشد و از آن‌جا که خیلی سرگرم بازی شده، از شما می‌خواهد تا جواب را به او بگویید.

---

## ورودی
در خط اول ورودی دو عدد صحیح `n` و `k` آمده‌ است.

```

1 ≤ k ≤ n ≤ 1000

```

---

## خروجی
در تنها خط خروجی، تعداد مراحلی را که طول می‌کشد تا دوباره نوبت حسنی شود چاپ کنید.

---

## مثال‌ها

### نمونه ۱
**ورودی**
```

5 2

```
**خروجی**
```

5

```
توضیح: اگر افراد دور دایره را از `1` تا `5` شماره‌گذاری کنیم به طوری که حسنی شمارهٔ `1` را بگیرد، مرحله‌ها به صورت `(1, 3, 5, 2, 4, 1)` تکرار می‌شود و دوباره نوبت `1` پس از 5 مرحله می‌شود.

### نمونه ۲
**ورودی**
```

6 2

```
**خروجی**
```

3

```
توضیح: افرادی که سلام می‌کنند به ترتیب `(1, 3, 5, 1)` هستند، پس پس از 3 مرحله نوبت حسنی می‌شود.

### نمونه ۳
**ورودی**
```

6 6

```
**خروجی**
```

1

```
توضیح: در این حالت نفر `k` تا بعدی حسنی خودِ حسنی است؛ پس بازی فقط ۱ مرحله طول می‌کشد.
