# یافتن عدد فوق اول

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۶۴ مگابایت

---

عدد **فوق اول** عددی است که هر زیرعدد سمت چپ آن نیز عدد اول باشد.  
به عنوان مثال:

- ۷۳۳۱ فوق اول است، زیرا:
  - ۷، ۷۳، ۷۳۳، ۷۳۳۱ همگی اول هستند.  
- اولین عدد فوق اول عدد ۲ است.

---

## ورودی

- یک عدد صحیح `n` — نمایانگر موقعیت عدد فوق اول مورد نظر.

```

1 ≤ n ≤ ?

```

---

## خروجی

- یک خط شامل عدد فوق اول `n`ام.

---

## مثال‌ها

### مثال ۱
**ورودی**
```

5

```

**خروجی**
```

23

```

**توضیح:**  
- اولین اعداد فوق اول: 2, 3, 5, 7, 23  
- عدد پنجم فوق اول 23 است.
