محدودیت زمان: ۱ ثانیه  
محدودیت حافظه: ۲۵۶ مگابایت  

اعداد صحیح `a` و `b` به شما داده می‌شود. از شما خواسته شده است که تعداد جواب‌های حقیقی معادله‌ی زیر را مشخص کنید:

```

ax + b = 0

```

- اگر این معادله **دقیقاً یک جواب** داشته باشد، عبارت `unique` را چاپ کنید.  
- اگر این معادله **بی‌شمار جواب** داشته باشد، عبارت `infinite` را چاپ کنید.  
- اگر این معادله **هیچ جوابی** نداشته باشد، عبارت `invalid` را چاپ کنید.

## ورودی
در سطر اول ورودی عدد صحیح `a` و در سطر دوم عدد صحیح `b` داده می‌شود.

```

-100 ≤ a, b ≤ 100

```

## خروجی
در تنها سطر خروجی یکی از سه عبارت زیر را چاپ کنید:

```

unique
infinite
invalid

```

## مثال‌ها

### ورودی نمونه ۱
```

2 -3

```

### خروجی نمونه ۱
```

unique

```

معادله به صورت `2x - 3 = 0` خواهد بود که دقیقاً یک جواب دارد (`x = 1.5`).

---

### ورودی نمونه ۲
```

0 7

```

### خروجی نمونه ۲
```

invalid

```

معادله به صورت `0×x + 7 = 0` است که هیچ جوابی ندارد.

---

### ورودی نمونه ۳
```

0 0

```

### خروجی نمونه ۳
```

infinite

```

معادله به صورت `0×x + 0 = 0` است که همهٔ اعداد حقیقی جواب آن هستند.
