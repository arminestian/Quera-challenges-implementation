# بررسی امکان ساخت مثلث با سه زاویه

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

باقر باید مشخص کند که آیا با سه زاویه داده شده می‌توان یک مثلث ساخت یا خیر.

---

## توضیح مسئله
یک مثلث معتبر باید شرایط زیر را داشته باشد:

1. مجموع سه زاویه برابر **۱۸۰ درجه** باشد.
2. هیچ زاویه‌ای نباید **صفر یا منفی** باشد.

اگر هر دو شرط برقرار باشند، می‌توان مثلث ساخت.

---

## ورودی
یک خط شامل سه عدد صحیح و نامنفی کمتر از ۳۶۰:

```

a b c

```

که نشان‌دهندهٔ درجهٔ سه زاویه هستند.

---

## خروجی
یک خط که شامل عبارت:

- `Yes` اگر مثلث قابل ساخت باشد
- `No` در غیر این صورت

---

## مثال‌ها

### ورودی نمونه ۱
```

70 60 50

```

### خروجی نمونه ۱
```

Yes

```
