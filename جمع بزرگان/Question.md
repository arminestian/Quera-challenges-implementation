# عملیات روی توان‌های ده

**محدودیت زمان:** ۰.۵ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

کیانوش برای عضویت در سازمان OC مصاحبه می‌دهد. در روز اول قرار است روی پشتکار او کار شود.

سؤال‌ها این‌گونه‌اند:  
عبارتی شامل دو عدد بزرگ و یک عملگر به شما داده می‌شود. اگر عملگر جمع (`+`) باشد باید **مجموع** آن دو عدد را خروجی دهید و اگر عملگر ضرب (`*`) باشد باید **حاصل‌ضرب** آن دو را چاپ کنید.

اما نکته‌ای که کیانوش فراموش کرده این است که در ادامه‌ی متن سؤال گفته شده بود **هر دو عدد دقیقاً توانی از ۱۰ هستند**؛ یعنی هر عدد را می‌توان به صورت `10^x` نوشت که `x` عددی صحیح و بین ۰ تا ۱۰۰ است.

شما باید برنامه‌ای بنویسید که با دریافت این دو عدد و عملگر، مقدار صحیح نتیجه را چاپ کند.

---

## ورودی
ورودی شامل **سه خط** است:

1. خط اول: یک عدد که توانی از ۱۰ است.  
2. خط دوم: یک کاراکتر `+` یا `*`.  
3. خط سوم: یک عدد که توانی از ۱۰ است.

تضمین می‌شود هر عدد برابر `10^x` است که:

```

0 ≤ x ≤ 100

```

---

## خروجی
در تنها سطر خروجی **نتیجهٔ عملیات** روی این دو عدد را چاپ کنید.

---

## مثال‌ها

### مثال ۱
**ورودی:**
```

10
+
100

```

**خروجی:**
```

110

```

**ورودی:**
```

10000

```
