# مسئله: چند جفت مربع؟

**محدودیت زمان:** ۱ ثانیه  
**محدودیت حافظه:** ۲۵۶ مگابایت

استاد به تازگی کتاب *ریاضیات و مکافات* را تألیف کرده‌است. در فصل اول این کتاب تعدادی پرسش به شکل «چند جفت مربع با اضلاع طبیعی، اختلاف مساحت `n` دارند؟» آمده که باید به آن‌ها پاسخ دهید.

---

## ورودی
در سطر اول ورودی عدد صحیح `t` تعداد پرسش‌های کتاب می‌آید.  
در هر یک از `t` سطر بعدی، یک عدد صحیح `n_i` آمده است که نشان‌دهنده‌ی اختلاف مساحت است.

```

1 ≤ t ≤ 1000
1 ≤ n_i ≤ 10^6

```

---

## خروجی
در سطر `i`ام از `t` سطر، پاسخ پرسش `i`ام را چاپ کنید — یعنی تعداد جفت‌های `(a, b)` از اعداد طبیعی با `a > b ≥ 1` طوری که:

```

a^2 − b^2 = n_i

```

---

## مثال

**ورودی نمونه ۱**
```

3
5
15
25

```

**خروجی نمونه ۱**
```

1
2
1

```

**توضیحات نمونه‌ها**

- برای `n = 5` تنها یک جفت وجود دارد: اضلاع `3` و `2` (۳² − ۲² = 9 − 4 = 5).
- برای `n = 15` دو جفت وجود دارد: `(4,1)` چون 4² − 1² = 16 − 1 = 15 و همچنین `(8,7)` چون 8² − 7² = 64 − 49 = 15.
- برای `n = 25` تنها یک جفت وجود دارد: `(13,12)` چون 13² − 12² = 169 − 144 = 25.
